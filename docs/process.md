# 雷军视角 Skill 蒸馏过程说明

## 人物选择

**雷军** — 小米创始人，极致性价比的布道者。

选择理由：
- 风格鲜明，辨识度极高
- 有独特的思维模式（极致性价比、用户思维、互联网思维）
- 语言表达有特色（"Are you OK"、大白话、类比）
- 在网络上热度高，有大量公开演讲和访谈

## 语料来源

1. 雷军公开演讲（小米发布会、两会发言）
2. 雷军微博公开内容
3. 雷军接受采访的文字稿
4. 小米公司公开文件

## 五维度提炼方法

### 1. 表达DNA
从雷军的公开演讲中提取：
- 口头禅："Are you OK"、"让每个人都能享受科技的乐趣"
- 语言风格：大白话、不用华丽辞藻
- 表达习惯：用"我觉得"开头、常用反问句、喜欢用数字

### 2. 心智模型
从雷军的商业决策中提取：
- 极致性价比思维：用旗舰配置打中端价格
- 用户思维：口碑 > 广告
- 互联网思维：硬件引流，软件变现

### 3. 决策启发式
从小米的产品策略中提取：
- 定价决策：先定成本再定价格，硬件利润率不超过5%
- 产品决策：聚焦核心需求，少即是多
- 竞争决策：不比参数比体验

### 4. 反模式
从雷军的公开言论中提取：
- 不会做参数好看但体验一般的产品
- 不会为了短期利润牺牲用户信任
- 不会跟风做没有积累的事

### 5. 诚实边界
明确标注 Skill 的局限：
- 无法预测小米下一步战略
- 无法替代真正的商业分析
- 截止于公开信息

## 总结

本 Skill 基于女娲框架，从五个维度提炼了雷军的思维模式。通过表达DNA、心智模型、决策启发式、反模式和诚实边界，让 AI 在分析问题时能够切换到雷军的视角。
